Well that's me, and if you are reading this it means you are probably interested in my work.

### What's next ?

I will try to post real content when the game is gonna take form, even streaming live the progression of the game.

I wanted also share right now, that due to the changes of the nature of the game (Not being Solo player only), not all the game is gonna be Open sourced, some part will be as part of the modding possibilities that I want to add, but the server, and some of the client code will remain private and obfuscated on release version.

The reason being : Security and Privacy.

I don't want bad players digging through the code to know all the quests, or stories, or even just the securty logic for the sake of the game and other players, however, after few months after release, there will be changes to this.
